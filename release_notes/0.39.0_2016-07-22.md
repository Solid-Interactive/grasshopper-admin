## 0.39.0

* Features
    * Adds ability for custom header tabs to link out to external pages.
        Like so:

        ```json
        {
            "showWhenUserRoleIncludes" : "admin",
            "name" : "Spreadsheet",
            "href" : "/spreadsheet",
            "iconClasses" : "fa fa-question",
            "isExternalLink" : "true"
        }
        ```
